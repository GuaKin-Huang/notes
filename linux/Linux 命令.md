- cd

|    **.**     | 代表此层目录                                   |
| :----------: | ---------------------------------------- |
|    **..**    | 代表上一层目录                                  |
|    **-**     | 代表前一个工作目录                                |
|    **~**     | 代表【目前使用者身份】所在的目录                         |
| **~account** | 代表 `account` 这个使用者的家目录（ `account`是这个账户的名称 ） |
